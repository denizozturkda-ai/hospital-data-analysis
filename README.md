# Hospital Performance Analysis Dashboard

This project analyzes hospital data to uncover insights related to patient costs, outcomes, readmission rates, and satisfaction.

## Project Overview

The analysis includes:

- Patient demographics (age, gender)
- Cost analysis per medical procedure
- Length of stay (average and maximum)
- Readmission rates
- Patient satisfaction based on outcomes

## Tools Used

- Python (Pandas & SQL in Jupyter Notebook)
- SQL queries for data extraction
- Power BI for dashboard visualizations

## Dashboard Features

- KPI cards (Total Patients, Average Cost, Satisfaction, etc.)
- Cost analysis by procedure
- Length of stay comparison (avg vs max)
- Readmission rate analysis per procedure
- Satisfaction by outcome
- Gender-based comparisons

## Files

- `Hospital-data-analysis.ipynb` → Data checking, SQL queries, and preparation of CSVs used for dashboard creation
- `csvs` → Processed data sets for each dashboard
- `hospital-data-dashboard.pbix` → Power BI dashboard using processed CSVs
  
## Key Insights

- Certain procedures have significantly higher costs
- Readmission rates vary across procedures
- Better outcomes are associated with higher patient satisfaction
- Gender differences exist in cost and length of stay

Created as a data analytics portfolio project to demonstrate data checking,  SQL usage, and Power BI dashboard creation using hospital datasets.
